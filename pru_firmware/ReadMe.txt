Code for Programmable Realtime Unit 0 and 1,
from librobotcontrol 
(https://github.com/StrawsonDesign/librobotcontrol)
Can be debugged using prudebug-rl.

PRU1 is 8 RC servo channels
PRU0 is currently reading quadrature encoder.

Plan is to modify PRU0 code for fast A/D read and clock signal generation
for line scan camera.
